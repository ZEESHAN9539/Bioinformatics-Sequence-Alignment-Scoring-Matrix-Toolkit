Bioinformatics Sequence Alignment & Scoring Matrix Toolkit

This project focuses on implementing core bioinformatics algorithms and sequence alignment techniques using Python. The notebook covers scoring matrix generation, global and local sequence alignment, and affine gap penalty implementation for protein sequence analysis.

Features
Amino acid frequency analysis
Scoring matrix calculations
FASTA file parsing
Global alignment using Needleman–Wunsch algorithm
Local alignment using Smith–Waterman algorithm
Affine gap penalty implementation
Protein sequence comparison using BLOSUM50 and BLOSUM62 matrices
Alignment score, identity, and similarity analysis
Technologies Used
Python
Bioinformatics algorithms
Dynamic Programming
FASTA sequence processing
BLOSUM scoring matrices
Project Highlights
Compared protein sequences from different species
Implemented both global and local alignment methods
Explored affine gap penalties for improved biological sequence alignment
Worked with real biological datasets in FASTA format
